## 快速开始

```bash
git clone https://github.com/DJChanahCJD/koala-nav.git

cd koala-nav

npm install
npm run dev

npm run build
```



## Cloudfare部署

1. 登录Cloudfare
2. 创建一个新的Pages项目
3. 连接到你的GitHub仓库
4. 配置项目设置
   - 构建命令：`npm run build`
   - 构建输出目录：`out`
5. 部署项目
